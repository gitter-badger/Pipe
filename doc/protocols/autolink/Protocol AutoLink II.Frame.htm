описание работыКаждый FRAME имеет постоянный размер равный 5 байтам, из которых первый байт обозначает номер переменной (всего 256 переменных), остальные 4 байта - значение этой переменной (см. табл. 1). структура переменной может быть разной, описание и метод упаковки приведен по каждой переменной

40TEMP_0Температурный датчик (1-wire) T_0 интервал температуры равен: от -500.00 до +500.00 C
на сервер температура передается умноженная на 100 (чтобы десятичные значения превратить в целое), например 28,33 будет 2833.
За 0 (ноль) градусов принимаем значение 50000, соот-но, при распаковке значения с устройства нужно отнимать 50000, например:
1. -26.33 преобразовываем -2633, прибавляем 50000 = 47367 передаем, на сервере отнимаем 50000 и опять получаем -2633
2. 16.55 преобразовываем 1655, прибавляем 50000 = 51655 передаем, на сервере отнимаем 50000 и опять получаем 1655

Т.е. диапазон по температуре должен быть от 0 до 100000.
Значение 111111 - говорит о неверном вычислении температуры

.
